# react-boilerplate-client-only

## Install
```
npm i
```

## Development
```
npm run start
```

## Build
```
npm run build
```

## Server
Server available on http://localhost:3000 when start task is run.

## Debug
Add `debug=app:*` to localStorage in inspector for enable console debuging.
