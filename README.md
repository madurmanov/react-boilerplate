# react-config

## Install
```
npm i
```

## Development
```
npm run start
```

## Production
```
npm run build
npm run start-prod
```

## Server
Server available on http://localhost:3000 when start task is run.

## Linter
Run linter before commit:
```
sh bin/hooks/install
```

## Debug
Add `debug=src:*` to localStorage in inspector for enable console debug.
