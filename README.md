# react-config

## Install
```
npm i
sh bin/hooks/install
```

## Development
```
npm run start
```

## Production
```
npm run build
npm run start-prod
```

## Server
Server available on http://localhost:3000 when start task is run.

## Debug
For enable debug in console add `debug=src:*` to localStorage in inspector.
