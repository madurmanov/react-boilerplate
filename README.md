# react-boilerplate-client-only

## Install
```
npm i
```

## Development
```
npm run start
```

## Build
```
npm run build
```

## Storybook
```
npm run storybook
```

## Debug
Add `debug=app:*` to localStorage in inspector for enable console debuging.
