# react-boilerplate

## Install
```
npm i
```

## Development
```
npm run start
```

## Production
```
npm run start:prod
```

## Storybook
```
npm run storybook
```

## Debug
Add `debug=app:*` to localStorage in inspector for enable console debuging.
